顧客カルテ

好み・注意点・メモを一人ずつ整理。

来店ログ

会った日・会話・次アクションを時系列で。

端末内保存

入力データは端末内で管理（外部送信なし）。

主な機能（無料で使える範囲）

顧客プロフィール

名前（ニックネーム）、仕事/属性、誕生日（任意）、メモなどを管理。

好み / NG の記録

「好き」「避けたい」を整理して、次の接客に活かせます。

タグ & 重要度

タグで分類、重要度で優先順位を把握。

来店（面談）ログ

日付・会話メモ・次アクションを顧客に紐づけて保存。

通知（任意）

誕生日/フォローの通知を端末内のローカル通知で設定できます（外部送信なし）。

オフライン利用

ネットが無い場所でも入力・閲覧が可能。

使い方

  1. 顧客を追加します（名前・タグ・重要度など）。
  2. 「好み / NG / メモ」を入力します。
  3. 来店（面談）後にログを追加し、会話内容と次アクションを残します。
  4. 次回の来店前に見返して、スムーズな対応につなげます。

例：好み：ウイスキー / カラオケ、NG：辛い話題、次回：誕生日の話をする

Pro（アプリ内課金 / 買い切り）

Proは任意の買い切り課金です。無料版でも顧客メモ・来店ログの基本機能は利用できます。 価格はApp Storeの商品ページでご確認ください。

画面ロック（端末パスコード）

アプリ復帰時に端末のパスコード認証で保護できます。
Face IDは使用しません（端末の認証方式に従います）。

高度な統計

期間フィルタ（7/30/90日・全期間）と、タグ別ランキングを表示します。

テンプレ（今後拡張）

お礼・次回提案のテンプレなど、運用をラクにする機能を順次追加予定です。

  • 購入はApp Storeの決済を通じて行われます。
  • 機種変更時はアプリ内の「購入を復元」をご利用ください。
  • 課金は任意です。無料版の基本機能が使えなくなることはありません。

スクリーンショット

顧客一覧：タグと重要度で見やすく。

カレンダー：来店ログを日付で管理。

統計：数値で顧客数、来店ログ、支出を管理。

プライバシー

おもてなしメモは、ユーザーが入力した情報（顧客メモ・タグ・来店ログ等）を端末内に保存して管理します。

  • 入力データの外部送信：なし
  • 第三者提供：なし
  • 分析SDK・広告SDK：なし
  • 通知：誕生日/フォロー等は端末内のローカル通知（許可した場合のみ）
  • 課金：App Storeの決済を使用（当サイト/アプリでカード情報等を取得しません）

データ削除：アプリを削除すると端末内データは削除されます。
